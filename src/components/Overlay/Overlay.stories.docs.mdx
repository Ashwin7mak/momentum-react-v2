The `<Overlay />` component. This component fully extends `<ModalContainer />`, but adds a dynamic interactive layer for accessibility. This is abstracted from `<ModalContainer />` to help improve performance when rendering `<ModalContainer />`s.

For additional information on how the WAI-ARIA in this component operates, view this [React-Aria documentation](https://react-spectrum.adobe.com/react-aria/useDialog.html).
